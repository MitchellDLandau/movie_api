Movie REST API I am building the server side functionality for a movie application. The application will provide users with information about movies, directors, and genres. Users will be able to sign up, edit their personal information, and create a list of their favorite movies. I have included a table below to help users know how the api will work with data so they can use it to be able to create their own apps. URL endpoints as well as any required information, methods, formats, and the responses expected format is included in the table. This application uses MongoDB and is currently set to work on port 8080. Make sure all required dependencies are installed including "body-parser": "^1.20.2", "express": "^4.18.2", "mongoose": "^7.3.4", "uuid": "^9.0.0" are installed. The MongoDB should be running with the name given to it ie. MovieDB and the node index.js should be run to start the server. An authorized moderator can be added by making the authorized in the database under authorization. If this is set to True they will be able to access the endpoints listed as (ADMIN ONLY) to be able to make more modifications to the data in the database. All users will have this setting automatically set to false when new accounts are made.
